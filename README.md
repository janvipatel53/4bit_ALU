# 4-bit ALU Design (Verilog)

This project is a 4-bit ALU made in Verilog.  
It performs different operations based on select lines.

## Features
- 4-bit inputs A and B
- Operations: add, subtract, AND, OR, XOR, NOT, shift left, shift right
- Carry output for add/sub
- Zero flag output
- Verified using testbench and waveform simulation
